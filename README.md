<br/> <b> Visualizer: </b> Glava

<br/> <b> Transparent top-bar: </b> Dynamic Panel Transparency
<br/> <b> Custom spotify theme: </b> Spicetify-cli
<br/> <b> Weather in top bar: </b> Meteo
